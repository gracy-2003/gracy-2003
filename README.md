## How to Use This Template

1. Create a new repository on GitHub with the exact same name as your GitHub username
2. Create a README.md file in that repository
3. Copy and paste the code above into the README.md file
4. Replace all placeholders:
   - Replace `your-username` with your actual GitHub username in all URLs and stats widgets
   - Replace `[Your Name]` with your actual name
   - Replace `[Your Location]` with your location
   - Replace `[Your Project Name]` with your project name and update the corresponding link
   - Replace `your-email@example.com` with your actual email
   - Replace `[Your Fun Fact]` with an interesting fact about yourself
   - Update social media links with your actual profiles
   - Add or remove tech stack icons as needed
